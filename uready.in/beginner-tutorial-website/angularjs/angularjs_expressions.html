    AngularJS Expressions

    Expressions are used to bind application data to html. AngularJS expressions can be written inside double braces: {{ expression }} Expressions behaves in same way as ng-bind directives. AngularJS application expressions are pure javascript expressions and outputs the data where they are used.

     

    Example

    <!DOCTYPE html>
    <html lang="en-US">
    <script src="http://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"> </script>
    <script>
              it('should calculate expression in binding', function() {
              expect(element(by.binding('1+2')).getText()).toEqual('1+2=3');
              });
    </script>
    <body ng-app = " " >
    <span>
         1+2={{1+2}}
    </span>
    </body>
    </html>

    AngularJS Numbers

    AngularJS numbers are like JavaScript numbers:

     

    Example

    <div ng-app = "" ng-init = "quantity=2;cost=6" >
    <p>Total Rupees: <span ng-bind = "quantity * cost"> </span> </p>
    </div>
    Run the file »

    AngularJS Strings

    AngularJS strings are like JavaScript strings:

     

    Example

    <div ng-app = "" ng-init = "firstname='vetri'; lastname='pandi'" >
    <p>The fullname is: {{ firstname + " " + lastname }} </p>
    </div>
    Run the file »

    AngularJS Objects

    AngularJS objects are like JavaScript objects:

     

    Example

    <div ng-app = "" ng-init = "person={firstname:'vetri', lastname:'pandi'}" >
    <p>The name is: {{ person.lastname }} </p>
    </div>

    AngularJS Arrays

    AngularJS arrays are like JavaScript arrays:

     

    Example

    <div ng-app = "" ng-init = "points=[2,30,50,60,80,90]" >
    <p>The second result is: {{ points[2] }} </p>

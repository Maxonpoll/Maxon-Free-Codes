    AngularJS Services

    Angular services are substitutable objects that are wired together using dependency injection (DI).

    You can use services to organize and share code across your app.

     

    Example

    <div id="simple" ng-controller="MyController">
        <p> Notify services for angular controller </p>
    <input ng-init="message='Hello'" ng-model="message">
    <button ng-click="callNotify(message);"> Notify me </button>
        <p> you have to click 3 times to see an alert </p>
    </div>
    Run the file »

    Creating Services

    Application developers are free to define their own services by registering the service's name and service factory function, with an Angular module. The service factory function generates the single object or function that represents the service to the rest of the application. The object or function returned by the service is injected into any component (controller, service, filter or directive) that specifies a dependency on the service.

    Registering Services

    Services are registered to modules via the Module API.

    Typically you use the Module factory API to register a service.

     
    var myModule = angular.module('myModule', []);
    myModule.factory('serviceId', function() {
            var shinyNewServiceInstance;         return shinyNewServiceInstance;
    });

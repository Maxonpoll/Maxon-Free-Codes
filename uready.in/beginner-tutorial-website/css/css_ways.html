    Three Ways to Insert CSS

    There are three ways of inserting a style sheet:

    1.External Style Sheet

    2.Internal Style Sheet

    3.Inline Style

    External Style Sheet

    With an external style sheet, you can change the look of an entire website by changing just one file. The style sheet file must be saved with a .css extension. Each page must include a reference to the external style sheet file inside the <link> element. The <link> element goes inside the <head> section:

     

    Example

    <head>
    <link rel="stylesheet" type="text/css" href="mystyle.css">
    </head>
    Run the file »

    mystyle.css file looks like

     
    h1 {
    color:black;
    letter-spacing:2px;
    text-align:center;
    }
    p {
    color:red;
    font-weight:bold;
    font-size:18px;
    }
    body {
    background-color:yellow;
    }

    Internal Style Sheet

    An internal style sheet may be used if one single page has a unique style.

    Internal styles are defined within the <style> element, inside the <head> section of an HTML page.

     

    Example

    <head>
    <style>
    h1 {
    color:yellow;
    letter-spacing:2px;
    text-align:center;
    }
    p {
    color:red;
    font-weight:bold;
    font-size:18px;
    }
    body {
    background-color:yellow;
    }
    </style>
    </head>
    Run the file »

    Inline Sheet

    An inline style may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

     

    Example

    <h1 style="color:blue;margin-left:20px;margin-top:10px;"> This is Inline Sheet </h1>
    <p style="color:red;letter-spacing:2px;font-weight:bold;"> This is a some example </h1>
    Run the file »
